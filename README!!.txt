Steps for using application:

1. Dump your Super Smash Bros Ultimate save file on Checkpoint
2. Make sure to name it "smash_stages".
3. Download your favorite smash bros custom stages
4. Place them in a folder on your sd card titled "Smash Maps" (case-sensitive). Also, make sure the map name doesn't have any 0's in the name.
5. Place "SmashMapInjector.nro" in the switch folder of sd card.
6. Run application.
7. Go to Checkpoint and restore the save.
8. Enjoy!
